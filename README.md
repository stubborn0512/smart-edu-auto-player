# 智慧教育平台自动连播脚本

一个用于自动关闭弹窗、自动播放、自动跳转下一视频的浏览器脚本。

## 功能

- 自动关闭弹窗
- 自动定位下一待学视频
- 自动播放视频
- 提升学习效率

## 使用方法

1. 安装 ScriptCat / Tampermonkey
2. 导入 `.user.js` 文件
3. 打开对应学习平台即可自动运行

## 适配环境

- Chrome / Edge
- ScriptCat / Tampermonkey
